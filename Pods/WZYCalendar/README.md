# WZYCalendar
#### 在iOS项目中一行代码快速集成一款精致的日历。（Objective-C）

<br>
## 相对于其他第三方而言的优势
* 解决数据加载过多内存消耗过大问题，一次只加载四个月份的数据。
* 解决控件循环引用问题，不会创建过多的控件，减轻界面显示时的压力。
* 提供回调接口，点击日期可以方便拿到当前的时间。
* 拒绝 xib、storyboard 搭建，纯代码编写。
* 项目中采用了模块封装，代码清晰，添加了比较多的注释，方便开发者的二次修改以及增加新的功能。
